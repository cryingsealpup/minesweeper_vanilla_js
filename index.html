Welcome to the Minesweeper game!

Here are some quick rules for playing Minesweeper:

  • Click on a square to reveal whether it contains a mine or not.
  • If you reveal a mine, the game is over.
  • If you reveal a cell without mine, the number on a cell shows the number of mines adjacent to it.
  • If you reveal a cell without heighbouring mines, it will expand opened area up to cells with some.
  • If you suspect that cell contains a mine, you can mark it with red flag by clicking on the right mouse button.
  • When all mines marked with red flag, the game is over.
